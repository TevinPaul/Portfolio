Photography Image Sharing Platform

VibeShare is a photography sharing platform where users can upload, browse and engage with images from a global community.

Project Goal & Overview

The goal of VibeShare is to create a platform where photographers can share, discover and engage with high quality images. This project aims to combine clean UI design with effective media handlng, offering a smoooth user experience across devices.

Key Features

User account creation and profile customization

Image upload with automatic resizing and optimization

Categorized and searchable photo galleries

Like and Comment functionality

Responsive, mobile friendly gallery layout

Secure authentication and data protection

Option to control image download permissions

Technologies Used

React.js

Tailwind CSS

Node.js

MongoDB

Cloudinary

JWT
